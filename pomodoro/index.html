Pomodoro Method Timer App

Info

This app was created with React, Javascript, HTML, & CSS
as an exercise for Free Code Camp.

With the completion of a work session + break session, the circle icons will be checked.

Chose the break length, session length, and session count with the arrows.

Max session count = 10, Max break length = 10, Max session length = 25

If you find a bug, let me know through github!
